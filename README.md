## How to run .js files

### On local
Make sure you have Node.js installed, then follow these steps:
- Open the terminal
- Change the directory to this project
- Use this command to run: **node "filename"**
    - Example: node 01-algorithm-oke-yes.js

### On an online compiler
- Open an online compiler like 
    - [Programiz](https://www.programiz.com/javascript/online-compiler/)
- Then, copy the source code of the .js files and paste it into the online compiler
